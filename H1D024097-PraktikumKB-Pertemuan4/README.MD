# Pertemuan 4 - Sistem Pakar 1

Repositori ini berisi hasil praktikum Mata Kuliah Kecerdasan Buatan (KB) Pertemuan 4 yang berfokus pada pengenalan Sistem Pakar menggunakan bahasa pemrograman Python (PySwip & Tkinter).

## Daftar File
Berdasarkan praktikum yang telah diselesaikan:
1. `silsilah.py` : Implementasi logika silsilah keluarga (Percobaan 1).
2. `malaria.py` : Program diagnosa malaria sederhana berbasis console (Percobaan 2).
3. `pakar_malaria.py` : Sistem pakar diagnosa malaria interaktif (Percobaan 3).
4. `pakar_malaria_gui.py` : Sistem pakar malaria berbasis GUI menggunakan Tkinter dan PySwip (Percobaan 4).
5. `kerusakan_komputer.py` : **[TUGAS MANDIRI]** Sistem pakar diagnosa kerusakan hardware PC/Laptop berbasis GUI.

## Fitur Utama
* **Logika Predikat**: Menggunakan fakta dan aturan untuk deduksi kesimpulan.
* **Basis Pengetahuan (Knowledge Base)**: Penyimpanan data gejala dan kerusakan menggunakan struktur Dictionary (Python) dan Klausa (Prolog).
* **Mesin Inferensi**: Mekanisme pelacakan gejala untuk menghasilkan diagnosa dan solusi singkat.
* **GUI Interaktif**: Antarmuka pengguna yang memudahkan input gejala melalui tombol Ya/Tidak.
