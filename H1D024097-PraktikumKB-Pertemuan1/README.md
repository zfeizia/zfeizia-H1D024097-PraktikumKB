# Tugas Praktikum Kecerdasan Buatan - Pertemuan 1

Program sederhana untuk mencatat *watchlist* drakor dan memilih satu judul secara acak untuk ditonton.

Menggunakan bahasa pemrograman python yang mengimplemetasikan konsep:
1. Struktur kontrol
* `while` 
* `for` 
* `if-elif-else` 

2. Struktur data
* Menggunakan struktur data `List` 

3. Library (minimal 2 library)
* `random`: Untuk memilih judul drakor secara acak dari list.
* `time`: Untuk memberikan efek jeda waktu saat program memilih drakor.
